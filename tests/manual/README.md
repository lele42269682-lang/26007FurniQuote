# 手工测试总指引

每个模块开发完成后，必须在此目录下创建 `{mid}_manual.md`，提供明确的手工测试步骤。

## 通用流程

1. 启动项目 `./start.sh` 或 `python master.py`
2. 浏览器打开 `http://localhost:8000/test`（测试 Dashboard）
3. 找到对应模块卡片，按 README 步骤执行
4. 在 Dashboard 上点击「✓ 确认通过」记录结果
